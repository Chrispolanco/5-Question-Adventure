# Mini Question Adventure

> This is an application with a Rails API and JS frontend. The app lets lets users take on 4 different stories  each with 5 questions. For each question there is only one right answer though. The user create their own account where they will be able to track their pogress and compare it with other users. 

**Installation**

- You will need Postgresql and a postgres server to use this application
- Clone this repository
- cd javascript-project
- cd javascriptproject-backend
- bundle install
- rails db:create
- rails db:migrate
- rails db:seed
- rails s
- Open javascriptproject-frontend/index.html in your browser
